---
## Описание 
Данная программа выводит случайную цитату. Можно выбрать автора(-ов) и тему(-ы). Если ничего не выбрать то выведется случайная из всех цитат. Через основное окно можно открыть дополнительные окна для добавления цитаты в бибилиотеку или просмотра истории цитат.
## Инструкция к использованию
Для корректной работы необходимо иметь в отдельной папке основной файл (Quote generator.py), дополнительный файл (Quote_adder.py) и json со списком цитат (Quotes list.json), который идет в комплекте.
Открываем и запускаем файл Quote generator, выбираем авторов из списка, жмем на кнопку "Выбрать авторов" (кнопки выбора сохраняют текущее состояние списков), аналогично со второй рамкой с выбором тем. Для вывода нажимаем на кнопку "Сказать что-нибудь". После использования программы в папке появится второй файл json (Quotes history.json), это запись историй, ее можно удалить (правда, она появится снова).
